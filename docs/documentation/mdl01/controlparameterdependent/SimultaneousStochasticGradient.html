SimultaneousStochasticGradient

ControlParameterDependent

This function solve a particular optimal control problem using the stochastic gradient descent algorithm. The restriction of the optimization problem is a parameter-dependent finite dimensional linear system. Then, the resulting states depend on a certain parameter. Therefore, the functional is constructed to control the average of the states with respect to this parameter. See Also in AverageClassicalGradient

Mandatory Input Parameters

Name Description class
iCPD Control Parameter Dependent Problem ControlParameterDependent
xt The target vector where you want the system to go double

Optional Input Parameters

Name Description class
tol tolerance of algorithm, this number is compare with $J(k)-J(k-1)$ double
beta This number is the power of the control, is define by follow expresion $$J = \min_{u \in L^2(0,T)} \frac{1}{2} \left[ \frac{1}{|\mathcal{K}|} \sum_{\nu \in \mathcal{K}} x \left( T, \nu \right) - \bar{x} \right]^2 + \frac{\beta}{2} \int_0^T u^2 \mathrm{d}t, \quad \beta \in \mathbb{R}^+ $$ double
gamma0 Length Step of the gradient Method. The control is update as follow $$u_{k+1} = u_{k} + \gamma \nabla u_{k}$$. In Stochastic method $\gamma_{k} = \gamma_0 * \frac{1}/{\sqrt{k}}$ double
MaxIter Maximun of iterations of this method double

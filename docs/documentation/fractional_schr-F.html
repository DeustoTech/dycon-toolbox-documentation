Solver the fractional Schrodinger equation in 1d

Input Vars

  • s : Order of the fractional Laplacian (-dx^2)^s
  • L : Define the interval of x, So x \in (-L,L)
  • N : number of points in the space discretization
  • T : length of the time interval
  • u0 : initial datum u0(x)

Output Vars

  • x : space partition
  • t : time partition
  • u : solution of the equation

Description

Solves the fractional Schrodinger equation

using FE for the approximation of the fractional Laplacian and Cranck-Nicholson for the time integration

For this example we choose following parameters

s = 0.5;
N = 100;
L = 1;

Execute the function

A = rigidity_fr_laplacian(s,L,N);

DyCon Toolbox

Welcome to the web interface of DyCon Toolbox, the computational platform developed within the ERC DyCon - Dynamic Control project.

DyCon Toolbox is a set of MATLAB classes, methods and functions which englobe the findings of research in the field of computational mathematics of the DyCon team.

Dycon Toolbox collects the different codes developed in the Dycon research project. This project develops cutting-edge theoretical results, however, transmitting these results to other researchers, not specialists in control theory, can be difficult. Even among experts on the subject of sharing software is complicated.

To do this, the DyCon toolbox has been created. Here is a series of tutorials for faithful and easy reproduction of mathematical results. The tutorials can be seen in the following link.

In addition, thanks to this compilation, it has also been possible to create different MATLAB classes and functions that allow creating new software content. These tools are documented in the following link.

The DyCon MATLAB Toolbox is primarily developed by the Chair of Computational Mathematics, in DeustoTech. However, we welcome any improvement or addition to the computational platform. Please contact the DyCon Toolbox Developer Team dycon-dev-group@deusto.es for further information.

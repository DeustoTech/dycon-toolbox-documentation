A wealth of information is available to help you understand, learn and use the DyCon Toolbox.

Below you will find the different classes and functions the DyCon Toolbox offers to help solve problems that may show up in real life TOFINISH

Classes

ControlProblem

This class is able to solve optimization problems of a function restricted to an ordinary differential equation. This scheme is used to solve optimal control problems in which the functional derivative is calculated. ControlProblem class has methods that help us find optimal control as well as obtaining the attached problem and it's derivative form, in both symbolic and numerical versions.

ControlParameterDependent

The Average Control class is able to calculate the average control given the matrices that depend on the parameter. It has as parameter the state where you want to take all the equations that depend on the parameters.

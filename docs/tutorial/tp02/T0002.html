En este tutorial mostraremos las tipos de descenso implementados para el metodos del gradiente para los problemas de control optimos

Creamos el objeto Optimal Control

Y = sym('y',[2 1]); U = sym('u',[1 1]);

define dynamics

 F = [ Y(2)             ; ...
      -Y(2)      + U(1) ] ;
%%
dynamics = ode(F,Y,U);
dynamics.Condition = [0;-1];
dynamics.dt = 0.01;

YT = [2;4];
Psi = sym(0);
L   =0.005*(U.'*U) +  (Y-YT).'*(Y-YT) ;
%% Creace de OptimalControl Object
iP = OptimalControl(dynamics,Psi,L);

GradientMethod ha sido definido como un metodo iterativo que actualiza el control dada un control anterior. Esta actualizaciï¿½n pretende mejorar el valor de $J(u)$ mediante un descenso del gradiente. Sin embargo se permite distintos tipos de descenso. Esta implementados tres tipos de descenso

  • ClassicalDescent: Se calcula el gradiente, $dJ$ dado un control y se actualiza el control de la siguiente forma: Siendo $\alpha$ una constante que puede ser modificada

  • AdaptativeDescent: Se actualiza el control de la misma forma que en ClassicalDescent, sim embargo el parï¿½metro $\alpha$ se multiplica por dos si el valor de J decrece, y se divide por dos si el valor de J crece. De esta forma, tenemos valores de $\alpha$ grande cuando el descenso nos lo permite, y valores de $alpha$, mas finos si estamos cerca de un mï¿½nimo local. - ConjugateGradientDescent: Se actuliza siguiendo el mï¿½todo de gradiente conjugado, porpuesto en [1]. Este actualiza el control siguiendo la direcciï¿½n del gradiente. El paso optimos $\alpha$, es calculado mediante optimizaciï¿½n numerica. Luego se actualiza el control con una combiancin lineal entre el gradiente en el punto actual y el gradiente en el punto anterior.
